BookKeeperNITJ

Bookkeepernitj is a web based application which helps students save money .There are many books that students want to read but can’t either due to unavailability or due to high price of books. Bookkeepernitj helps students get these books. Users can signup with google and upload details of books available with them. Users in need of these books can search for the availability on the web app and can contact the book lenders directly. Once the book is lended the availability status gets changed and it becomes no longer available to public. CRUD operation can be performed on uploaded details. It uses Bootstrap for FrontEnd and Express for BackEnd. Other libraries used include Passport for authentication, Mongodb for managing the database etc.


PathFinder

PathFinder is a react based visualization of Dijkstra’s Algorithm which finds the shortest path between the source node and the destination node in a grid, besides the presence of wall node in the path .In this visualization one can place wall nodes anywhere in the grid and the algorithm always finds out the shortest path.
